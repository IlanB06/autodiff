# How to run
Clone the repo:
```sh
git clone git@github.com:IlanB06/autodiff.git 
cd autodiff
```
Sync uv:
```sh
uv sync
```
Activate venv:
```sh
source .venv/bin/activate     # Linux/macOS
# or
.\.venv\Scripts\activate      # Windows
```
Run the project:
```sh
python main.py
```
